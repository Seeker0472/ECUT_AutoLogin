# AutoLogin
c语言实现东华理工大学校园网自动登录

>注意事项
>>1.如果想实现开机自动登录，请自行添加计划任务  
>>2.计划任务程序最好勾选使用最高权限运行  
>>3.目前只适配了无线连接，有线连接正在适配  
>>4.目前只适配了中国移动，其他运营商等待测试
